补充

为了保证 ip 地址的安全性和可用性

　　1. 在 linux 的 /etc/hosts 文件中配置 内网ip 地址和主机名的映射环境，这样在shell脚本或者代码中使用主机名替代 ip 使用

　　2. 如果是云主机，在windows中的hosts文件中配置外网ip地址和主机名的映射

注意：hosts文件中的前两行切记不能删，否则可能带来bug
